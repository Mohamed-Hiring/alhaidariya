السلام عليكم ورحمة الله وبركاته

تم تطوير نظام حديث لإدارة تأجير المعدات الثقيلة. هذا النظام يتطلب:

✅ إعداد Supabase (قاعدة بيانات)
✅ نسخ النظام على GitHub Pages
✅ إنشاء حسابات المستخدمين

================================================================================
STEP 1: حمّل الملفات من GitHub
================================================================================

المستودع: https://github.com/Mohamed-Hiring/alhaidariya

الملفات المهمة:
• index.html — الملف الرئيسي للتطبيق
• cards-data.js — بيانات البطاقات الأولية
• SYSTEM_GUIDE.md — شرح النظام كاملاً
• IT_SETUP_GUIDE.md — دليل التسطيب خطوة بخطوة
• README.md — معلومات تقنية

================================================================================
STEP 2: اقرأ التوثيق
================================================================================

ابدأ بـ: IT_SETUP_GUIDE.md

يحتوي على:
1. إعداد Supabase (مع SQL scripts)
2. تحديث الكود بالمفاتيح
3. نشر على GitHub Pages
4. إنشاء حسابات
5. اختبار النظام

================================================================================
STEP 3: لاحظ النقاط التقنية
================================================================================

• الملف الرئيسي: HTML واحد + ملف بيانات منفصل
• قاعدة البيانات: Supabase (PostgreSQL)
• الأمان: Authentication عبر Supabase + RLS
• الاستضافة: GitHub Pages (مجاني)
• المتصفح: أي متصفح حديث

================================================================================
STEP 4: التسطيب
================================================================================

المدة المتوقعة: 30-45 دقيقة

1. Supabase Setup: 10 دقائق
   - إنشاء account
   - نسخ المفاتيح
   - تشغيل SQL script

2. Code Setup: 5 دقائق
   - تحديث المفاتيح في index.html

3. GitHub Setup: 5 دقائق
   - Fork المستودع
   - تفعيل Pages

4. User Creation: 10 دقائق
   - إنشاء حسابات المستخدمين
   - إضافة بيانات ملفاتهم

5. Testing: 10 دقائق
   - تسجيل الدخول
   - إنشاء بطاقة تجريبية
   - اختبار الحفظ

================================================================================
STEP 5: من يتصل بك إذا واجهت مشاكل؟
================================================================================

اقرأ قسم "استكشاف الأخطاء" في IT_SETUP_GUIDE.md

أكثر المشاكل الشائعة:
• "Cannot load the app" → تحقق من المفاتيح
• "Sign In doesn't work" → تحقق من قاعدة البيانات
• "Data not saving" → تحقق من RLS policies

================================================================================
STEP 6: بعد النسطيب
================================================================================

✅ ادخل الموقع من رابطك الجديد
✅ سجّل الدخول بحساب التجربة
✅ جرّب إنشاء بطاقة جديدة
✅ تأكد من الحفظ في Supabase
